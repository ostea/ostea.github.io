MAC更新导致git异常

发表于 2018-01-12 |

问题

今天早上在更新mac系统的后，AS打开出现异常:
can’t start git: /usr/bin/git
probably the path to git executable is not valid . fix it.

原因

mac系统更新导致，通常默认的git都是和一个开发工具command line tools相依赖，假如是iOS/mac开发者随着xcode 一起升级就很少出现这事情，其他语言开发者比如android ，因为第三方ide依赖系统git 也就是command line tools，可能随着系统更新导致路径变更。

解决途径

方法一

  • 打开终端terminal
  • 执行 xcode-select —install
  • 安装xocode 或者 下载command line tools工具即可

方法二

去git官网下载，并将AndroidStudio里的Git路径重新指向即可，这也是一劳永逸的。即使以后Mac版本变更也不太可能出现这个问题(推荐).
